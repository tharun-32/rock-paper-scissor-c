# CLI Rock Paper Scissor game Program in C

## Description
It is a CLI based rock paper scissor game where list of string with rock paper scissor is provided
and random function is used to pick one and compare it with user input 
the one to reach 2 points first wins

## Concepts Learned 
I have Learned how to copare string using strcmp 
and how to assign string to a initilised string using strcpy function
